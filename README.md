## Desafio
O desafio consistia em, criar 2 Microserviços, um para Produtos, onde podemos registrar e requisitar os produtos, onde é feito a persistência de dados em um banco de dados H2 e, outro para os pedidos, que consome do primeiro microserviço e não necessitava de persistência de dados, um servidor Eureka e um Api Gateway para roteamento de acesso.
